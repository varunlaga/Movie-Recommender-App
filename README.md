## About
This is a data-driven web application that provides personalized movie recommendations. The system uses Item-Based Collaborative Filtering (IBCF) to suggest movies based on user ratings and falls back to a popularity-based system for new profiles.

## Get Started
1. Clone the repo: git clone https://github.com/varunlaga/Movie-Recommender-App.git

2. Install dependencies: pip install pandas numpy requests streamlit

3. Run the app: streamlit run recommender_app.py
